# 📋 Quick Reference Card

## 🚀 Essential Commands

```bash
# Start Server
npm start

# Stop Server
Ctrl + C

# Install Dependencies (first time)
npm install
```

## 🌐 Important URLs

- **Menu**: http://localhost:3000
- **Admin**: http://localhost:3000/admin.html

## 📁 Key Files

- `database.json` - Your menu data (BACKUP THIS!)
- `server.js` - Backend server
- `public/` - Frontend files
- `uploads/` - Product images

## 🔑 Admin Features

### Add Product
1. Go to admin panel
2. Fill form
3. Upload image (max 5MB)
4. Click "Add Product"

### Edit Product
1. Find product in table
2. Click "Edit"
3. Update fields
4. Click "Update"

### Delete Product
1. Click "Delete" button
2. Confirm deletion

### Backup Data
1. Click "Export Data"
2. Save JSON file

### Import Data
1. Click "Import Data"
2. Select JSON file
3. Confirm replacement

## 🌍 Deploy Online

### Easiest: Render.com
1. Sign up at render.com
2. Create "Web Service"
3. Connect GitHub
4. Deploy!

### Fast: Railway.app
1. Sign up at railway.app
2. Deploy from GitHub
3. Auto-deploys!

## 🆘 Quick Fixes

**Website not loading?**
→ Check terminal, restart with `npm start`

**Products not showing?**
→ Check database.json exists

**Can't upload images?**
→ Max 5MB, JPG/PNG/GIF/WEBP only

**Changes not appearing?**
→ Refresh browser, check server running
